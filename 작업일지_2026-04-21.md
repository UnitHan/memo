### 1. 재시작마다 "메모 복원 완료" 팝업 뜨는 문제

- **수정**: `App.tsx` — `initDataBackup()` 복원 성공 시 팝업 표시 코드 제거, 콘솔 로그만 출력하도록 변경

```ts
// 변경 전
if (restored) {
  setAlertModal({ isOpen: true, title: '✅ 메모 복원 완료', ... });
}

// 변경 후
if (restored) {
  console.log('✅ 이전 메모 데이터 자동 복원 완료');
}
```

### 2. 보관함(⋯) 버튼 클릭 시 창 백화현상
- **원인**: JS(`new WebviewWindow(...)`)로 settings 창 생성 시 `additional_browser_args`가 적용되지 않음 → `tauri://` 스킴에서 WebView2가 비보안 컨텍스트로 인식 → 렌더링 실패(백화)
- **수정**: settings 창 생성 로직을 Rust Command로 이전, main 창과 동일한 `additional_browser_args` 적용

**`src-tauri/src/lib.rs`**
- `use tauri::Manager` 추가
- `open_settings_window` Tauri Command 신규 추가
  - 기존 settings 창이 있으면 포커스만
  - 없으면 `WebviewWindowBuilder`로 창 생성 + Windows에서 `additional_browser_args` 동일 적용
- `invoke_handler`에 `open_settings_window` 등록

```rust
#[tauri::command]
async fn open_settings_window(app: tauri::AppHandle) -> Result<(), String> {
    if let Some(window) = app.get_webview_window("settings") {
        window.set_focus().map_err(|e| e.to_string())?;
        return Ok(());
    }
    let builder = tauri::WebviewWindowBuilder::new(&app, "settings", ...)
        ...
    #[cfg(target_os = "windows")]
    let builder = builder.additional_browser_args(
        "--unsafely-treat-insecure-origin-as-secure=tauri://localhost http://tauri.localhost"
    );
    builder.build().map_err(|e| e.to_string())?;
    Ok(())
}
```

**`src/App.tsx`**
- `openSettingsWindow()` 함수를 `invoke('open_settings_window')`로 교체
- 불필요해진 `WebviewWindow` import 제거

```ts
// 변경 전: new WebviewWindow('settings', { url: fullUrl, ... })
// 변경 후
const openSettingsWindow = async () => {
  await invoke('open_settings_window');
};
```

### 3. 보관함 창 라우팅 방식 개선
- **원인**: 기존 hash 라우팅(`#settings`)은 JS로 창 생성 후 URL fragment를 직접 지정했으나, Rust에서 창을 생성하면 hash를 넘길 방법이 없음
- **수정**: `src/main.tsx` — hash 기반 → **창 라벨(windowLabel) 기반** 라우팅으로 변경 (hash 폴백 병행 유지)

```ts
// 변경 전
const hash = window.location.hash.split('?')[0];
{hash === '#settings' ? <SettingsPage /> : <App />}

// 변경 후
const windowLabel = getCurrentWindow().label;
{windowLabel === 'settings' || hash === '#settings' ? <SettingsPage /> : <App />}
```
